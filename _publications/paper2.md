abstract: >-
  We collect eviction records to test whether court assignment effects the default eviction probability in LA County. Using a regression discontinuity design, we find gaps in default probability near courthouse boundaries. Mechanisms (tenant costs and court procedures) are explored using variation across boundary segments.
